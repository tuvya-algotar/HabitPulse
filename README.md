# HabitPulse 🚀

**HabitPulse** is a beautifully crafted, local-first habit tracking web application designed to help you manage your daily routines without any mental overload. Focused on extreme simplicity, privacy, and visual excellence, HabitPulse features a sleek, premium dark-mode aesthetic with Apple-level fluid animations.

![HabitPulse Hero](public/og-image.png)

## ✨ Features

- **Premium Deep Dark UI**: A meticulously constructed monochrome aesthetic featuring pure blacks, minimalist white accents, and carefully calibrated typography.
- **Fluid & Dynamic Animations**: Powered by `framer-motion`, every interaction tells a story from smoothly expanding reminder cards to dynamic progress rings.
- **Smart Reminders**: Set specific alarms for medicine, hydration, items, and routines. Easily edit, delete, or snooze reminders right from the dashboard.
- **Progress Tracking**: Visualize your day with an elegant, minimalist progress ring and streak counters to keep yourself accountable.
- **Local-First & Privacy-Focused**: No signups, no accounts, and no backend databases required. All of your data and preferences persist entirely locally on your device via `localStorage`.

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router, React 18+)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Components**: Radix UI primitives & custom UI modules
- **Language**: TypeScript

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tuvya-algotar/HabitPulse.git
   cd HabitPulse
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the App:**  
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## 🤝 Philosophy

HabitPulse was built around **Human-Centered Design**. Every UI decision was made to reduce friction. It's not about making a giant list of tasks; it's about making a seamless, beautiful system that guides you through your day effortlessly. 

Enjoy building better habits!
